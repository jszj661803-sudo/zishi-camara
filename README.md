# AI 拍摄指导相机 H5 真机体验版

这是第一阶段验证版，目标是先跑通：

- 手机摄像头实时取景。
- 人物姿势辅助线。
- 根据模板推荐姿势参考图。
- 根据画面亮度、清晰度、背景状态给出评分和问题提示。
- 人物美化入口：背景虚化、瘦脸、美白、美肤、瘦腰、拉长腿、丰胸、彩妆。

## 当前验证库

- 已接入 100 张青年女性参考图。
- 参考图只用于内部验证姿势规则，不作为正式产品公开素材。
- 后续确认流程后，再把参考图统一生成或重绘成我们的标准模特图。

## 本机体验

在本目录启动网页服务后，电脑浏览器打开：

```text
http://127.0.0.1:8016/
```

## 手机真机体验

手机和电脑连接同一个 Wi-Fi 后，用手机浏览器打开电脑局域网地址，例如：

```text
http://电脑IP:8016/
```

注意：部分手机浏览器对摄像头权限要求更严格。如果手机无法打开摄像头，先用电脑浏览器验证页面流程，再把 H5 部署到 HTTPS 地址测试。

## 后续接真实 AI 的位置

- `app.js` 里的 `sampleFrame()`：当前只做亮度和清晰度分析，后续可接场景识别、人像分割、姿态识别。
- `data/pose-library-100.json`：当前是 100 张验证库，后续可扩展为完整姿势数据库。
- `assets/pose_refs/`：当前是内部参考缩略图，正式版要替换为标准模特素材。
